THS Instagram PSD README
-----------------------------------
Font is Noto Sans (https://fonts.google.com/specimen/Noto+Sans)
36pt bold for headers and event names (24pt alt for long event names)
18pt regular for location and date

Colors are THS Orange (255,102,51), Black (0,0,0), and white (255,255,255)
Text backings are 80% opacity and have an 8px gaussian blur behind on the 
background image.
